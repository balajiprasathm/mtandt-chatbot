This folder is where you add extra knowledge for the chatbot.

HOW TO ADD A DOCUMENT:
1. Open this "knowledge" folder in your mtandt-chatbot repo.
2. Click "Add file" -> "Upload files".
3. Drag in a .txt or .pdf file (product brochures, price lists, FAQs, policies, etc).
4. Commit the changes.
5. Render will automatically redeploy and the bot will read the new file the next
   time it starts up.
